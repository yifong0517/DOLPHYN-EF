# MESS.jl
Julia package for macro energy synthesis system modeling involving multi energy sectors like power orginated from [GenX](https://github.com/GenXProject/GenX) and hydrogen originated from from [DOLPHYN](https://github.com/macroenergy/DOLPHYN).

This repo is used to gather multi energy sectors into a more modularized and organized way of modeling for easier energy system modeling. Later development is lined up for carbon capture and utilization as well as bioenergy.
